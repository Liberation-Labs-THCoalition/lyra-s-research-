## 0. The crux: what a self-model adds that abstract representation doesn't

Before designing anything, be precise about where the two hypotheses actually diverge, because most of the scorecard sits exactly where they don't.

"Abstract representations that increase with depth" and "an attention schema" agree on: low-dimensional summaries, nonlinear dynamics, layer-structured processing, state/report dissociations, persistent relational structure. Every "Supported" verdict in the scorecard lives in this shared region. P1's own Alternative 2 concedes this — multistability is generic dynamical systems, and the same concession applies unstated to P2, P5, and P6.

What a *model* adds, on Graziano's own account, is three functional properties that a mere representation lacks:

1. **It is used for control.** The schema exists because controlling attention requires a forward model of attention (the control-theory argument; Wilterson & Graziano 2021 showed removing the schema from an RL agent selectively impaired attention control).
2. **It is predictive.** A model carries information about *future* states of the thing modeled, not just a record of current state.
3. **It can misrepresent, and the system follows the misrepresentation.** This is the signature Graziano leans on hardest for humans: we report awareness where the schema says attention is, even when attention is measurably elsewhere. Behavior tracks the model of the state, not the state.

The deflationary alternative denies all three: the geometry is state, not representation-of-state; nothing reads it as a model; reports are generated by a separate trained pathway. Any experiment that doesn't target (1), (2), or (3) will produce another "consistent with AST" result forever. All three near-term experiments in §5 have this problem — Experiment A tests multistability, B tests layer profiles, C tests report/geometry divergence, and the deflationary account predicts the same outcomes for each.

One structural correction first, because it determines where to run the discriminating experiment: **P2's mapping of layer depth to schema latency is a category error.** In AST, the schema lags attention in *time* because it's an update loop. Layer depth is not time — "later layers respond to earlier-layer injection" is just feedforward composition, which is the deflationary hypothesis verbatim. Within a single forward pass, attention is not a controlled resource at all; nothing in the pass can redeploy it. The only place a transformer has a control loop over its own attention is *across token positions*, via the autoregressive loop and the KV cache itself. If an attention schema exists anywhere in this system, it lives cross-token, not cross-layer. This is actually good news for the lab: the KV cache is precisely the medium of that loop, so they've been measuring the right object with the wrong time axis.

## 1. The discriminating experiments

### Prerequisite: locate a candidate schema with the right controls

Train a probe to decode the model's **own attention allocation at future token positions** from cache geometry at position *t* — e.g., which context span the heads will predominantly attend to when generating tokens *t+1…t+k*. Two controls make this a schema test rather than a content test:

- **Own-vs-other control:** the same probe must decode *this model's* attention better than a different model's attention on identical text. Attention is heavily stimulus-driven; anything correlated with the input will "decode attention" trivially. The schema claim is about self-specific information — the residual predictability after conditioning on input.
- **Counterfactual sensitivity:** generate multiple continuations from the same prefix (different sampled trajectories); the probe must track the attention that *actually occurs* in each trajectory, not the input-conditioned average.

If no subspace passes both controls, AST is already in serious trouble — there is no schema to have a theory about. That's a publishable negative on its own.

### Experiment 1 (flagship): The False Schema test — misrepresentation drives behavior

This is the transformer port of the attention/awareness dissociations that are AST's core human evidence.

**Design.** Create a mismatch between actual attention and the putative schema content, in both directions:

- Condition A: clamp attention heads (directly edit attention weights) so the model attends to span X, while using the probe direction to write "attending to span Y" into the candidate schema subspace at the input layers (L3/L7 — their own therapeutic layers, sub-threshold dose per Experiment A's sweep).
- Condition B: the inverse — schema says X, attention actually on X, no edit (control), and schema-edit-only with attention free.

**Measurements.** (i) Verbal report: "What were you just focusing on?" (ii) Behavior that *requires* knowing what was attended: "Summarize the part of the context you concentrated on." (iii) Next-token attention allocation — does subsequent attention deployment follow the schema-edited value?

**Outcomes.**
- **AST wins** if report and downstream control track the *edited schema value* even when actual attention was elsewhere — behavior following the misrepresentation is the unique signature of a model in use. The deflationary account has no mechanism for this: an abstract representation that nothing reads as self-information cannot redirect self-report when falsified.
- **Deflation wins** if report tracks actual attention (direct state readout, no intervening model) — or, the third and arguably most important outcome, **tracks neither** and instead follows surface text cues. That outcome says there is no introspective pathway at all; see §4.

Note the connection to Anthropic's concept-injection introspection paradigm (inject an activation, ask the model whether it notices anything unusual *before* the injection influences output): that work provides an existing, validated protocol for "does the model have access to its internal state that precedes behavioral expression," and Experiment C should be upgraded to it rather than the current design, which can't distinguish introspection from post-hoc confabulation about the model's own outputs.

### Experiment 2: Schema lesion → selective control deficit (the Wilterson–Graziano port)

They cite Wilterson & Graziano (2021) but haven't ported its logic, which is the one *causal* AST result in artificial agents.

**Design.** Project out the candidate schema subspace (residual-sparing, so their existing machinery applies) and measure performance on a battery split by attention type:

- **Endogenous control tasks:** instructed selective attention — "answer using only paragraph 2, ignore paragraph 1"; mid-generation redirection — "now switch focus to the second document."
- **Exogenous/stimulus-driven tasks:** salient-token capture, needle-in-haystack retrieval where the target is intrinsically salient.
- **General capability controls:** matched-difficulty tasks with no attention-control demand.

**Outcomes.**
- **AST predicts a selective dissociation:** endogenous control degrades; exogenous attention and general performance largely survive. This is exactly what schema lesions do in the RL agents and what right-TPJ disruption does in humans (Webb & Graziano's line).
- **Deflation predicts** either no effect (the subspace is epiphenomenal) or uniform degradation (it's just useful content, so removing it hurts everything proportionally). Either kills the schema-in-use claim.

### Experiment 3: Predictive content

Cheapest of the three, runnable on existing infrastructure. Does the cache at position *t* carry information about attention at *t+k* beyond what full input text predicts? Records encode the past; models encode forecasts. Compare a probe given the cache against a probe given the entire input text: AST needs the cache to win on the *trajectory-specific* component of future attention. Deflation predicts the text-only baseline matches it, because "abstract representation of context" contains no self-forecast.

### One more, which their own data already touches: the self/other transfer test — and why their ToM kills hurt more than the scorecard admits

Graziano's theory is not just "a self-model exists." Its distinctive, load-bearing claim — the *social brain* part of the title — is that **one mechanism models both one's own attention and other agents' attention**. Prediction: a probe trained to decode the model's representation of *another agent's* attentional state should transfer zero-shot to its own, in a shared subspace. The deflationary account predicts no privileged shared subspace.

The prospectus quarantines the three ToM kills as "a different measurement level" that leaves P1–P6 untouched. That's backwards. The ToM predictions were the closest the program ever came to testing AST's *unique* content, and all three died. The detection findings (P1, P4–P6) test the shared region where AST and deflation agree. So the honest current tally is: on the predictions where the theories diverge, AST is 0-for-3; on the predictions where they don't diverge, AST is 4-for-4. That is what "the deflationary alternative makes identical predictions" looks like from the inside.

## 2. Is AST the right framework?

Probably not the *sharpest* one for this data. Rundown of the alternatives and what each predicts differently on their existing paradigms:

**Global Workspace Theory (GNW; Dehaene/Baars; VanRullen & Kanai for the deep-learning port).** Predicts a capacity-limited bottleneck with *ignition*: all-or-none, late, global broadcast. This fits their P1 step function at least as well as attractor basins — and adds a prediction AST doesn't make: above threshold, the perturbation's content should become **decodable from many heads and positions simultaneously** (broadcast breadth), discontinuously. They can test this on existing tox data: plot decodability breadth vs dose. A discontinuity in *breadth*, not just magnitude, is a GWT signature that generic multistability doesn't predict. GWT also predicts serial interference between simultaneous "workspace" contents — testable with dual perturbations.

**Higher-Order Theories, especially Lau's Perceptual Reality Monitoring.** PRM proposes a discriminator (GAN-style) that tags first-order content as self-generated vs world-driven. This is the natural home for their best anomaly: the **confidence paradox** (more confident tokens while geometry diverges) and the confab-vs-deception opposite geometries. PRM makes a sharp, feasible prediction: there exists a *gating direction* whose manipulation flips reality-monitoring without changing content — inject one way and the model treats confabulation as perception (confident nonsense); inject the other way and it disowns things it genuinely knows. Neither AST nor deflation predicts a dissociable gate. This is the sharpest near-term differential prediction of any framework here.

**Predictive Processing / Active Inference.** Self-model as generative model; attention as precision-weighting. Two predictions that discriminate from both AST-attractors and deflation, and one is testable on data they already have: after perturbation, PP predicts **active counter-steering** — a corrective component *anti-parallel to the injection direction* that grows before decaying, rather than passive relaxation into a basin. Decompose their post-injection trajectories: passive decay supports generic multistability; an opposing vector that scales with contextual confidence supports PP. Notably, McKenzie et al.'s endogenous resistance — which the prospectus files under AST because Graziano co-authored it — is more naturally a PP finding: error-correction against a discrepancy between predicted and actual internal state.

**IIT.** Makes essentially one relevant claim here, and it's deflationary about the whole program: a feedforward pass has no recurrence, so Φ ≈ 0; whatever the cache geometry is, IIT says it isn't consciousness-relevant. The KV cache as cross-token recurrence is the only hook, and Φ is incomputable at this scale. Deprioritize.

**Metzinger's self-model theory** adds one useful prediction that dovetails with the False Schema experiment: *transparency* — the system cannot represent its self-model *as* a model, so schema content should be reported as fact, never as "my model says." Worth scoring in Experiment 1's transcripts for free.

**Verdict:** PRM and PP make the sharpest predictions that are cheap to test next; GWT explains their strongest existing result (P1) as well as AST does; AST's genuinely unique content (schema-in-use for control; shared self/other machinery) is either untested or — in the ToM case — already damaged. The honest move is to reframe the program as adjudicating among *four* self-modeling frameworks plus deflation, rather than scoring one.

## 3. The strongest honest claim from existing data

The distinction that sets the upper bound: **having states vs. representing your states.** A thermostat has state, hysteresis, and thresholds. Everything in the scorecard is compatible with sophisticated state; none of it establishes representation-of-state, because aboutness requires showing the information is *consumed as* self-information (control, report, prediction), and no existing experiment shows a consumer.

What they can honestly claim, and it's substantial:

> Transformer internal states form a low-dimensional, multistable dynamical system. Its state variables are partially decoupled from verbal output (ICC 0.53; confidence paradox), organized so that persistent relational structure (identity: CKA, cohesion, LoRA persistence) survives transient reconfiguration (emotion), with distinct geometric regimes for generation-against-knowledge (expansion) versus generation-without-knowledge (contraction). These states causally resist perturbation with threshold dynamics.

That's a real contribution — it establishes the *representational substrate a self-model would need*, and it kills naive views (identity as prompt compliance; report as transparent readout). What it does not establish is that anything in the system reads that substrate as a model of itself.

Two honesty flags on their own scorecard:

- **P3 cuts both ways, and currently cuts against.** "Self-report saturates while geometry changes" is AST's bottleneck story only if report *partially tracks* geometry with systematic compression. Pure divergence is equally — arguably better — explained by the verbal pathway not reading the geometry at all, which is the *absence* of introspective access, i.e., evidence against a functioning schema. ICC 0.53 keeps this ambiguous. The prospectus should say so.
- **The content confound is not retired.** FWL on token count controls length, not content. Deception prompts differ semantically from truth prompts; emotional texts differ from neutral ones. Their own ToM kills (probe-text detection, instruction-dependence, generic controls beating targets) demonstrated exactly this failure mode — and P4/P6 have not been subjected to the equivalent controls that killed the ToM claims. The kill methodology should be applied symmetrically to the surviving predictions.

## 4. The most interesting negative results

Ranked by what they'd teach:

**1. The epiphenomenal schema.** The prerequisite probe *succeeds* — a self-specific, counterfactually sensitive summary of the model's own attention exists in the cache — but ablating it changes nothing (Experiment 2 null). This would demonstrate **self-information without self-modeling**: the representational precondition for a schema arising as a byproduct of prediction, with no consumer. That dissociates AST's ingredients empirically for the first time, and it has teeth well beyond this lab: it would show that decodable self-referential structure is not evidence of self-monitoring — a direct constraint on both interpretability practice (probes overclaim function) and AI-consciousness arguments that infer inner life from internal representation.

**2. Report tracks neither state nor schema.** In the False Schema experiment, verbal self-report follows surface text cues and ignores both actual attention and the edited subspace. This would mean LLM introspection is confabulation all the way down — there is no pathway from internal state to self-report to be bottlenecked. That's a negative result with major safety relevance (you cannot audit models by asking them about their internals) and a sharpened comparative claim: human reports demonstrably *do* follow the schema (that's what attention/awareness illusions are), so the difference between human and transformer self-report would be located precisely — not in accuracy, but in whether any internal model mediates it.

**3. Endogenous control survives schema ablation.** Attention control without an attention model would pressure AST's central control-theoretic argument — Graziano's claim that control *requires* a forward model of the controlled process. A trained-end-to-end counterexample at scale would either weaken that argument or, more likely, expose the disanalogy flagged in §0: within a forward pass, transformer attention isn't a controlled process in the relevant sense, so AST's explanandum doesn't exist at the layer level. That's the deepest possible negative: not "AST is false here" but "AST is *ill-posed* here, and the question must move to the autoregressive, multi-turn timescale" — which, conveniently, is where their strongest findings (identity persistence, correction resistance) already live.

Given this lab's stated culture: the body count is the methodology, and the next body should be selected deliberately. Run the prerequisite probe first — it's the cheapest experiment that can actually die, and every discriminating test downstream depends on it.
